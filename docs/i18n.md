# Internationalisierung (i18n)

Dieser Ordner enthält alle Übersetzungsdateien für die Anwendung.

## Struktur

```
i18n/
├── de.php              # Deutsche Übersetzungen
├── en.php              # Englische Übersetzungen
├── fr.php              # Französische Übersetzungen
└── messages/           # Gruppierte Übersetzungen
    ├── de/
    │   ├── common.php
    │   └── errors.php
    └── en/
        ├── common.php
        └── errors.php
```

## Dateiformat

Übersetzungsdateien sind PHP-Arrays:

```php
<?php
return [
    'welcome' => 'Willkommen',
    'greeting' => 'Hallo {name}!',
    'items_count' => 'Sie haben {count} Artikel',
    'date_format' => 'Datum: {date}',
];
```

## Verwendung

### Übersetzungen laden

```php
$template = new Template();
$i18nModule = $template->getModule('i18n');

// Methode 1: Direkt nach Locale laden
$i18nModule->loadLocale('de'); // Lädt i18n/de.php

// Methode 2: Spezifische Datei laden
$i18nModule->loadTranslations('messages/de/common.php', 'de');

// Methode 3: Manuell setzen
$i18nModule->setTranslations([
    'key' => 'Wert'
], 'de');
```

### Im Template verwenden

```html
<!-- Einfache Übersetzung -->
<h1>{{i18n:welcome}}</h1>

<!-- Mit Variablen -->
<p>{{i18n:greeting|name:John}}</p>

<!-- Mit dynamischen Werten -->
<p>{{i18n:items_count|count:5}}</p>
```

## Locale und Zahlenformatierung

Die Zahlenformatierung wird automatisch basierend auf der Locale angepasst:

- **de, de_DE, de_AT, de_CH**: Dezimal `,` Tausender `.`
- **en, en_US, en_GB**: Dezimal `.` Tausender `,`
- **fr, fr_FR**: Dezimal `,` Tausender ` ` (Leerzeichen)
